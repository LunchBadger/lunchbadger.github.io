Open Source Technology Used in LunchBadger

LunchBadger Canvas Objects are wired into elements of open source software. Source-level access to these components provides opportunities in portability and environment-specific testing. Here are some projects used behind the scenes and integrated into LunchBadger:

  • Express Gateway - Provides advanced gateway features using Service Endpoint, Gateway, and API Endpoint Objects.
  • Loopback - Provides features of Data Source and Model Objects.
  • Kubernetes - Provides advanced container orchestration.
  • Serverless - Acts as an adapter for Function-as-a-Service (FaaS) platforms.
  • Kubeless - A FaaS solution that’s native to Kubernetes. This project powers LunchBadger’s out-of-the-box serverless platform.
